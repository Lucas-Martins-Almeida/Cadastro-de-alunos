📚 Sistema de Cadastro de Alunos em Python

Este projeto é um programa desenvolvido em Python com o objetivo de cadastrar alunos e analisar o desempenho da turma.

Ele foi criado como um exercício prático para aplicar conceitos fundamentais da programação.

🚀 Funcionalidades

O sistema permite:

✅ Cadastrar vários alunos
✅ Armazenar:
Nome
Idade
Nota
✅ Validar entradas do usuário
✅ Classificar alunos automaticamente:
🟢 Aprovado (nota ≥ 7)
🟡 Recuperação (5 ≤ nota < 7)
🔴 Reprovado (nota < 5)
✅ Calcular a média da turma
✅ Exibir relatório completo com:
Lista de alunos
Situação de cada aluno
Média da turma
Quantidade de aprovados, recuperação e reprovados
Maior nota
Menor nota
🛠️ Conceitos Utilizados

Este projeto utiliza diversos conceitos importantes:

Variáveis
input() para entrada de dados
Conversão de tipos (int, float)
Estruturas condicionais (if, elif, else)
Estruturas de repetição (while, for)
Funções
Listas
Dicionários
Tratamento de erros (try / except)
Validação de dados
📋 Regras de Validação
A idade deve ser maior que 0
A nota deve estar entre 0 e 10
O nome deve conter apenas letras
O programa trata erros e solicita nova entrada quando necessário
